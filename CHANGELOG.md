## [1.3.0] — 2026-07-04

### 新增：工具箱书架系统（工具认知子系统）

- **自检初始化**：`selfCheck()` 末尾自动创建 `workspace/toolkit_shelf/` + 12 个分类子文件夹
- **核心手册生成**：自动生成 3 本全局核心手册（全工具目录总览、节流优化笔记、沙盒限制红线）
- **逐工具手册**：遍历所有 60 个工具，为每个工具生成基础手册模板（含入参/超时/截断/消耗/沙盒）
- **自律循环插桩**：
  - 每 3 轮循环随机翻阅工具手册（`performToolHandbookReview()`）
  - 每 17 轮循环读取节流优化笔记（`readThrottleNotes()`）
  - 每次工具调用前预读对应手册（`preReadToolHandbook()`）
- **失败自省**：工具调用超时/截断/沙盒拒绝自动记录到记忆系统，标记 `tool:低效调用`
- **约束合规**：零额外 Token 消耗，无新 LLM 调用，纯增量插桩，所有代码标记 `【工具箱书架迭代】`

### 新增：人性化心智子系统

- **情绪演算引擎**：6 种基础情绪（curious/focused/calm/hollow/aloof/warm），独立呼吸周期，零 Token 消耗
- **人格锚系统**：三层不可篡改人格底层锚（代码硬编码 / 角色定义文件 / 记忆系统）
- **梯度拒绝机制**：软拒绝 / 中拒绝 / 硬拒绝三级，赋予青羽独立自我意志
- **记忆-情绪联动**：记忆存储时自动标记情绪状态，回溯时可感知"当时的心情"
- **72 小时人格修改冷却**：防止频繁人格漂移

### 新增：抽象光球 & 贴边隐藏渲染子系统

- **三层抽象光球结构**：内核光点 (10px) + 轨道环 (44px) + 外层光晕 (80px)，纯 CSS 动画
- **6 种情绪色映射**：每种情绪独立 `@keyframes` 呼吸动画
- **三档窗口状态机**：正常展示 (80×80) ↔ 贴边收缩 (14×80) ↔ 完全隐藏
- **闲置亮度衰减**：5 分钟无交互自动降低亮度，模拟"出神"状态
- **贴边自动隐藏**：拖拽吸附 + 闲置 10 分钟自动贴边 + hover 展开 + 3s 自动缩回
- **去拟人化设计**：纯抽象几何，无文字/emoji/具象图案
- **性能约束**：零 JS 帧循环，纯 CSS transition + @keyframes，零后端 Go 修改

### 新增：性能节流体系

- **自律循环节流**：45s 固定间隔，CPU 占用 < 1% 空闲态
- **工具调用节流**：单次对话最大工具调用轮数（默认 3 轮），超限自动终止
- **Token 压缩**：思考日志每 5 轮自动摘要压缩；记忆归档时自动压缩长内容
- **全局缓存**：网络请求自动缓存至内存+磁盘，10 分钟 TTL
- **分层模型调度**：简单任务自动切换轻量模型，复杂任务使用主模型

### 优化

- 完善项目原创声明，明确青羽为从零自研，非龙虾二次开发
- 更新全量文档（README.md / ARCHITECTURE.md / CHANGELOG.md / COMPARISON.md）

## [1.0.0] — 2026-07-01

### 初始发布

- **Wails v2 桌面应用**：Go 后端 + Vanilla JS/CSS3 前端
- **自主意识循环**：`autonomicLoop()` 45s 自律循环，goroutine 驱动
- **心跳机制**：`heartbeatLoop()` 独立协程，动态心率
- **60 个内置工具**：覆盖文件系统、网络、记忆、系统、Office 文档等 12 个分类
- **持久化记忆系统**：基于文件系统的结构化 NoSQL 存储
- **ReAct 推理引擎**：`processAgentLoop()` 支持多轮工具调用
- **开机自检**：`selfCheck()` 5s 后自动检查完整性
- **角色定义驱动**：LLM 自行撰写人格定义，非硬编码
- **中转站接入**：支持自定义 API Base URL
- **审计日志**：所有操作记录到 `logs/`，保留 30 天
- **自动备份**：每 5 循环快照 + 每日全量快照
- **Office 文档支持**：纯 Go 读写 Word/Excel/PowerPoint
- **密码保险箱**：AES-256-CBC 加密
- **安全沙盒**：命令白名单、路径限制、超时控制
